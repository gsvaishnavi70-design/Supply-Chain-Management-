# Supply-Chain-Management-
Analyzed supply chain data using Excel, SQL, Power BI, and Tableau to optimize inventory, demand forecasting, and logistics performance. Built interactive dashboards and reports to track KPIs such as delivery time, stock levels, and supplier efficiency. Derived actionable insights to improve operational efficiency and reduce costs.
